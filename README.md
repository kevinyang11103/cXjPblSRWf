# 课程管理系统简介 java1164

## 项目概述

课程管理系统是一款基于B/S架构和MVC模式的Web应用程序，旨在提供便捷的课程管理服务。系统采用前后端分离的设计，后端使用Java和SpringBoot技术栈，前端使用Vue框架。

## 技术栈

- 后端：Java、SpringBoot、Mysql
- 前端：Vue
- 开发工具：IDEA、JDK1.8、Maven、Mysql8.0

## 功能模块

### 系统角色

系统包含三个角色：管理员、教师和学生。

#### 管理员模块

- 首页
- 系管理
- 专业管理
- 办理管理
- 学生管理
- 教师管理
- 课程管理
- 选课修改
- 管理员管理

#### 教师模块

- 首页
- 授课查询
- 教师课表
- 成绩查询

#### 学生模块

- 首页
- 选修课程
- 学生课程
- 课表查询
- 成绩查询
- 信息维护
